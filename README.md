# static site template

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Compiles and minifies and watch for production
```
yarn watch
```

### Lints
```
yarn lint
```

### Lints and fixes files
```
yarn lint:fix
```
